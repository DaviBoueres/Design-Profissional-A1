# Design-Profissional-A1# 
DesignProfissional_A1  
Projeto desenvolvido por **gustavruas** como parte da avaliação final da disciplina de Design.  

---

## Propósito  
Este repositório reúne a maior parte das atividades realizadas ao longo do semestre, evidenciando não apenas domínio técnico, mas também dedicação, pesquisa e desenvolvimento de **soft skills** essenciais para a formação acadêmica e profissional.  

---

## Projetos e Objetivos  

### Calculadora em C  
- Aplicação prática de lógica de programação: condicionais, loops, funções e definição de variáveis.  
- Desenvolvimento de autonomia por meio de pesquisa sobre funções matemáticas e sua implementação.  

### Sistema Gerencial de Alunos  
- Exploração de conceitos fundamentais de **Programação Orientada a Objetos**:  
  - Classes  
  - Objetos  
  - Funções  

### Estudo de Algoritmos de Ordenação (Sort)  
- Compreensão teórica e prática de algoritmos de ordenação.  
- Implementação de diferentes métodos para análise de eficiência e aplicabilidade.  

### Projeto001  
- Primeiros passos no desenvolvimento web com:  
  - **HTML**  
  - **CSS**  
  - **JavaScript**  

### Projeto002  
- Aplicação do framework **Materialize** para criação de interfaces modernas e responsivas.  

### Prova A1 Grupo 07  
- Utilização do **Bootstrap** para construção de páginas dinâmicas e funcionais.  
- Implementação de recursos como navbar fixa e mapas interativos.  

### Cidade Inteligente  
- Desenvolvimento de modelos conceituais e lógicos com **BrModelo**.  
- Criação de modelo físico em **PostgreSQL** para representar uma cidade inteligente.  
- Integração entre teoria de modelagem e prática em banco de dados.  

---

## Tecnologias Utilizadas  

### Linguagens e Ferramentas  
- **Markdown**: documentação e criação de READMEs.  
- **C**: projetos de lógica e algoritmos (bibliotecas `math`, `stdio`, `stdlib`).  
- **HTML, CSS, JavaScript**: desenvolvimento web básico.  

### Frameworks  
- **Materialize**: design responsivo e moderno.  
- **Bootstrap**: sites dinâmicos com múltiplos recursos visuais e interativos.  

### Modelagem e Banco de Dados  
- **BrModelo**: modelagem conceitual e lógica.  
- **PostgreSQL**: implementação do modelo físico para o projeto Cidade Inteligente.  

---

## Conclusão  
Este repositório é a consolidação de um semestre de aprendizado, prática e evolução. Cada projeto representa um passo no desenvolvimento de competências técnicas e pessoais, reforçando o compromisso com o conhecimento e a aplicação criativa das ferramentas estudadas.
